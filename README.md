# FlowGuard
Designed an intelligent traffic management system using dual YOLOv5s models for real-time vehicle detection and emergency vehicle prioritization. Automated traffic light distribution based on road demand calculations at intersection points to optimize traffic flow and emergency response times.

## Structure

- **arduino/**: Traffic controller code (`.ino`)  
- **python/**: Python scripts for detection and main logic  
- **models/**: Pretrained models for AI detection  
- **assets/**: Diagrams and screenshots  

## Requirements

- Python packages: Listed in `requirements.txt`
- Arduino IDE for traffic controller
